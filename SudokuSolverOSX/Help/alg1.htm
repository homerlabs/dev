Algorithm 1- Mono Cell

Main Page

This algorithm can be performed on rows, columns, and blocks. Using rows for an example, this algorithm checks on each row for an occurrence of a value being found in only one cell. If such cells are found then those cells are considered Solved at this point.

The before and after screenshots below show this applied to rows:

Notice on row 1 that the vlaue '7' only appears once. Therefore it is that cell where the '7' must be placed. Same thing for row 2, value '9' and for row 5, value '7'.

When this algorithm is applied to this particular puzzle in column mode, no such cells are found.
