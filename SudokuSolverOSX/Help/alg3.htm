Algorithm 3- Mono Sector

Main Page

This algorithm can be performed on rows and columns. Each row has 3 sectors. A sector contains 3 consecutive columns that reside within the same block. For example, rows 1, 2, and 3 each have 3 cells in block 1 (columns 1, 2, and 3), 3 cells in block 2 (columns 4, 5, and 6), and 3 cells in block 3 (columns 7, 8, and 9).

Notice on row 1 that the vlaue '7' only appears once. Therefore it is that cell where the '7' must be placed. Same thing for row 2, value '9' and for row 5, value '7'.

When this algorithm is applied to this particular puzzle in column mode, no such cells are found.
