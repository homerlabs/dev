Sudoku Solver Data File Format

Main Page

The text file consists of 82 lines of text. The first line contains the puzzle name and the remaining 81 lines contain puzzle cell data in the form of 2 single characters separated by a 'tab'.

The 'value' for the cell will be the solution (1-9) or 0 if the cell is unsolved.

The 'status' will be a value between 0 and 3 inclusive. Status values: Given = 0, Unsolved = 1, Changed = 2, and Solved = 3.

The format can be verified by saving any puzzle and inspecting the text with a text editor.
