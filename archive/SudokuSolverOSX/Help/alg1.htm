Algorithm 1- Mono Cell

Main Page

This algorithm can be performed on rows, columns, and blocks. Blocks is disabled for this mode because after running this algorithm for rows and columns, no more information will be gained by running on blocks. Using rows for an example, this algorithm checks on each row for an occurrence of a value being found in only one cell. If such cells are found then those cells are considered Solved at this point.

Notice on row 6 that the vlaue '8' only appears once (column 2). Therefore it is that cell where the '8' must be placed. Same thing for row 8, value '8'. Column 2 was solved for the value '8' so all other '8's on this column where prunned (rows 1 and 3).

When this algorithm is applied to this particular puzzle in column mode, no such cells are found.
