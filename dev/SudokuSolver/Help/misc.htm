Unsolved Node Count

The unsolved node count is calculated by counting all the numbers in all the cells then subtracting 81 as there will be 0 unsolved nodes when there is only 1 number in each cell. When the unsolved node count reaches 0 the puzzle is solved and the Solve button becomes disabled.

The puzzle is solved by repeated application of the three algorithms available through the Algorithm Selection bar just under the puzzle cell array.

All 3 Algorithms Perform Calculations on Rows

To perform an operation on a Column, the 9x9 matrix is transposed, the operation is performed, and then the matrix is transposed back. To perfrom an operation on a Block, the Block is mapped to a Row, the operation is performed, then the results are remapped back to a Block.
